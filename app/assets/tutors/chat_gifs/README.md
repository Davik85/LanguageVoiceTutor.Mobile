# Chat-mode tutor GIFs

This folder is reserved for future lesson chat header GIF assets.

## Scope

- Mode: normal lesson chat header
- Layout target: wide upper-header visual area
- Tutors: `lana`, `nelli`, `david`
- Desktop-aligned core states: `idle`, `listening`, `thinking`, `speaking`, `transcribing`

## Recommended size

- Aspect ratio: `16:9` preferred
- Supported alternate ratio: `4:3`
- Recommended export size: `1280x720`
- Safe center area: keep the face and upper torso inside the middle `960x540`

## Naming

Use lowercase snake case:

- `lana_idle.gif`
- `lana_listening.gif`
- `lana_thinking.gif`
- `lana_speaking.gif`
